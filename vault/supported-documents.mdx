Vault has been fine-tuned on a broad range of business-critical document types. When you upload a document, Vault identifies its type and then extracts a set of predefined tags — questions the AI answers about that specific document. This page lists every supported document type and the tags Vault will attempt to extract from each.

For all document types, Vault also extracts **company and signatory information** in addition to the type-specific tags listed below.

<Note>New document types are added on a regular basis. If you need a document type that is not listed here, contact the TextMine team at support@textmine.com.</Note>
