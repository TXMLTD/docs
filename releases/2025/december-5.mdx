---
title: "TextMine — 5 December 2025 release"
description: "Adds chained tag reasoning, anchor tags for page placement, an improved image model, bulk Legislate-to-Vault transfers, tag counters, and a summary beta."
---

The 5 December 2025 release brings reasoning to chained tags, a new anchor tag feature, an improved image model, bulk document transfers, tag counters, and an early-access document summary.

## Chained tags now provide reasoning

The reasoning allows you to ask more complex questions over extracted data.

## Anchor tags

If you would like a tag to be connected to a different page, you can move it with the anchor icon.

![Anchor icon on a tag card](https://downloads.intercomcdn.com/i/o/h2osv3w3/1867639587/288908b387e23bc43aed2d89fd8e/Screenshot+2025-12-05+at+11_15_34.png)

You can then scroll or navigate to the page where you would like to place the tag.

![Navigating to the target page for the anchor tag](https://downloads.intercomcdn.com/i/o/h2osv3w3/1867638466/084b318f75f3b43e7ee53671f3a3/Screenshot+2025-12-05+at+11_15_46.png)

This will set the tag to the corresponding page and this change will be reflected in both the page navigation and PDF export.

## Additional improvements

The release also includes a number of small improvements.

**Improved image model**

The image model is now able to analyse larger images with more complex tasks.

**Bulk send documents from Legislate to Vault**

![Bulk send documents from Legislate to Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1867757578/ffd1f5ba8a20cedcb0f0aba0ecb3/Screenshot+2025-12-05+at+12_20_58.png)

**Tag counters are now displayed in the tags section**

![Tag counters in the tags section](https://downloads.intercomcdn.com/i/o/h2osv3w3/1867756302/62f2814f6d7251853e4d6ed719fd/Screenshot+2025-12-05+at+12_19_56.png)

**Summary (beta)**

Generate a summary of a document. This functionality is a beta version intended for informational purposes and cannot yet be used in PDF or CSV exports. It will be improved significantly over the coming weeks.

![Document summary beta feature](https://downloads.intercomcdn.com/i/o/h2osv3w3/1867764134/5a5e2e486e1a2b714a09cd43cf6c/Screenshot+2025-12-05+at+12_23_22.png)
