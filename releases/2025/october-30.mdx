The 30 October 2025 release adds native SharePoint integration so you can import files directly from SharePoint into Vault.

## SharePoint integration

It is now possible to import files natively from SharePoint. To do this you will need the following information about your SharePoint from your SharePoint/Azure administrator:

- Client ID
- Tenant ID
- Client Secret
- Site URL

Save these details in the **Vault Settings** of the Vault you would like to connect to TextMine.

![SharePoint credentials in Vault settings](https://downloads.intercomcdn.com/i/o/h2osv3w3/1804665263/47eee8e97d4d3950268aa6a3aecb/Screenshot+2025-10-30+at+08_03_12.png)

Test and then save the configuration.

![Testing and saving the SharePoint configuration](https://downloads.intercomcdn.com/i/o/h2osv3w3/1804666532/5de3b67d91c40e13d5c51c4b5d90/Screenshot+2025-10-30+at+08_35_10.png)

This will link the SharePoint with the Vault, meaning you will be able to import files from SharePoint into this Vault.

To do this, click **Upload document** and select **Use an integration**.

![Upload document with Use an integration option](https://downloads.intercomcdn.com/i/o/h2osv3w3/1804669348/29627678859ad7f71520b223385d/Screenshot+2025-10-30+at+08_35_25.png)

The SharePoint option will appear if the settings have been configured.

![SharePoint option in the integration picker](https://downloads.intercomcdn.com/i/o/h2osv3w3/1804670046/f1c30f3b6b108fd501222abaa1ec/Screenshot+2025-10-30+at+08_35_34.png)

Select which documents you would like to import.

![Selecting documents from SharePoint](https://downloads.intercomcdn.com/i/o/h2osv3w3/1804671690/7378af98c94ebabaf717299df933/Screenshot+2025-10-30+at+08_35_47.png)

Documents will then be imported into the Vault and go through the usual extraction process.

![Documents imported from SharePoint processing in Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1804670710/a24b15fe792735dd2cd6aacfc9e5/Screenshot+2025-10-30+at+08_36_04.png)
