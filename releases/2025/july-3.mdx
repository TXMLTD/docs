The 3 July 2025 release adds column selection to CSV exports in Legislate, giving you control over exactly which data fields are included in each export.

## CSV export improvements

It is now possible to customise which columns to export to CSV. After selecting which documents to export in Legislate, a pop-up will appear asking you to confirm which columns to include.

![Column selection pop-up before CSV export](https://downloads.intercomcdn.com/i/o/h2osv3w3/1601275855/b06fb496457bb545c4cb1f950e26/Screenshot+2025-07-03+at+10_41_37.png)

Once exported, the CSV will only contain the columns you selected.
