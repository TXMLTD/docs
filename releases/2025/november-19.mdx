---
title: "TextMine — 19 November 2025 release"
description: "Adds image tags, chained tags, folder sidebar improvements, Vault editor updates, fuzzy company search, localised timezones, and folder-scoped agent actions."
---

The 19 November 2025 release introduces image tags, chained tags, folder enhancements, Vault editor improvements, fuzzy company search, localised timezones, and folder-scoped agent actions.

## Folder improvements

Folders are now displayed in the sidebar below the business unit.

![Folders displayed in the sidebar](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836717586/b421a320cc838b3cacae59bda35c/Screenshot+2025-11-18+at+09_19_41.png)

In addition to filtering by folder, you can now sort the Vault view by folder.

![Sorting the Vault view by folder](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836722843/2f6c8efc8e81299857758387e4a9/Screenshot+2025-11-18+at+09_20_59.png)

## Vault editor improvements

The Vault editor's performance and usability have been improved.

When validating tags, you will be taken to the next tag to be validated automatically.

When a tag presents data in a list format, a list editing tool will be suggested to make handling the data easier.

![List editing tool suggestion in the Vault editor](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836732480/4f1db9ee3eb2f82acd2a7f9ec90c/Screenshot+2025-11-18+at+09_24_16.png)

You can search, keep, or delete the selection.

## Image tags

You can now prompt images within documents. Click on **Create image tag** and select the portion of the image you would like the model to analyse.

![Selecting an image region for an image tag](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836743686/eff652e71e244c596c9f1883b2d2/Screenshot+2025-11-18+at+09_27_59.png)

Once you have selected the image, create and process your question.

![Creating a question for the selected image region](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836746836/fcf29b05e7d2dc7d96481ce914ac/Screenshot+2025-11-18+at+09_32_21.png)

You will then see the answers as image tags in the editor.

![Image tag answers displayed in the editor](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836752012/f6cd252d188577909515be3d96a7/Screenshot+2025-11-18+at+09_35_08.png)

## Chained tags

A chained tag is a new tag type that allows you to answer questions using answers from other tags. To create a chained tag, click on **Custom tags** and enable the chained tag option.

![Enabling the chained tag option](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836759750/66ee3567916ec2d39ffd18a3ef99/Screenshot+2025-11-18+at+09_38_45.png)

This will open a pop-up with tags you can use as source tags for your question.

![Source tag selection pop-up](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836761725/735229438a126b131d4e5209db43/Screenshot+2025-11-18+at+09_39_40.png)

Once you have selected the source tags, create your question.

![Creating the chained tag question](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836761996/0c7578037263ba4907e7ec4b4a8b/Screenshot+2025-11-18+at+09_39_45.png)

Once you run the chained tag, the answer will display with a chain icon.

![Chained tag answer with chain icon](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836763182/fe2b3f6efe02b67b5ffc0a8ef03f/Screenshot+2025-11-18+at+09_37_40.png)

You can run chained tags on any type of tag, including previously run chained tags.

## Company search improvements

Fuzzy logic is now applied to company name searches to overcome typos, punctuation differences, or abbreviations.

![Fuzzy company search results](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836785956/8dc8cd91b6d2686064591ec3cc1b/Screenshot+2025-11-18+at+09_49_45.png)

![Fuzzy search matching an abbreviated company name](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836785518/9e4ed4f68955c1d76e566b9b73ba/Screenshot+2025-11-18+at+09_49_53.png)

## Timezones

Times in the app are now displayed based on the user's browser timezone.

<Note>
  Times in PDF exports are still based on the server's timezone.
</Note>

## General actions

General agent actions can now be applied to folders. This allows the agent to answer questions by looking at the extracted data from all documents in a folder.

![Applying a general action to a folder](https://downloads.intercomcdn.com/i/o/h2osv3w3/1836817160/40de84c37173d4be67d52f043abd/Screenshot+2025-11-18+at+09_54_27.png)
