The 16 September 2025 release introduces agent actions, enabling agents to assign or submit documents automatically based on Vault confidence scores and configurable conditions.

## Agent actions

Agents can now perform actions based on Vault results. The first action is based on the confidence score of a document and allows you to assign or submit it based on the score.

To get started, go to an Agent with an existing source and select **Add Action**.

![Add Action button in the agent configuration](https://downloads.intercomcdn.com/i/o/h2osv3w3/1728492780/2cab15f2c54675b7da56daeacb3f/Screenshot+2025-09-16+at+13_57_30.png)

You can determine whether the agent should run on existing documents or only on new documents uploaded to the connected Vault. Configure the conditions and actions to suit your requirements.

![Action conditions and settings configuration](https://downloads.intercomcdn.com/i/o/h2osv3w3/1728494358/a32095e72e96c8f8c4678257c296/Screenshot+2025-09-16+at+13_57_53.png)

You will see a summary of the results once the agent has finished running.

![Agent action results summary](https://downloads.intercomcdn.com/i/o/h2osv3w3/1728497579/f402bf1562516db6f0d64cebfc1d/Screenshot+2025-09-16+at+14_04_53.png)

<Tip>
  The list of available actions is expanding to include the ability to apply rules against values extracted by Vault. Get in touch if you would like to discuss your use case and how it can be solved by TextMine's agents.
</Tip>
