---
title: "TextMine — 25 September 2025 release"
description: "Adds Form ADV downloads from the SEC via Vault integrations, folder organisation in Legislate, and a tabular data view for documents in the platform."
---

The 25 September 2025 release introduces Form ADV downloads via Vault integrations, folder support in Legislate, and a grid/tabular view for document data.

## Form ADV and Vault integrations

Form ADVs can now be directly downloaded from the SEC via TextMine Agents or Vault's new integration page. When you click **Upload Document**, there is a new option to use an integration.

![Upload Document with the Use an integration option](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743837684/885d3412958ca947dac3eb8ee5a8/Screenshot+2025-09-25+at+11_54_42.png)

You can then select which integration you would like to use.

![Integration picker showing available integrations](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743838543/b806f2adc898090c2589f204370b/Screenshot+2025-09-25+at+11_54_49.png)

Once selected, specify the companies whose Form ADVs you wish to download.

![Company search for Form ADV download](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743839132/d354ecc0b87a9fd6a9e60c59c2dc/Screenshot+2025-09-25+at+11_55_08.png)

You will then have the option to validate which companies you wish to download.

![Company validation step before download](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743839552/a459b1088fce13a59714e4312c7c/Screenshot+2025-09-25+at+11_55_20.png)

Once selected, the download will start and the documents will eventually be processed by Vault.

![Form ADV download in progress](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743842533/2bfcd6978742bc60a2f04cce673b/Screenshot+2025-09-25+at+11_55_28.png)

![Form ADV documents processing in Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743842843/8a358148d475e630e6f536462896/Screenshot+2025-09-25+at+11_55_54.png)

## Legislate folders and tabular data views

Documents in Legislate can now be organised in folders. You can move documents between folders by selecting them.

![Folder organisation in Legislate](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743850592/5e7a464e5aa7dbcfd14db86b2fe5/Screenshot+2025-09-25+at+12_04_54.png)

A tabular view of data within the platform is also now available. Simply switch to the grid view.

![Grid/tabular view in Legislate](https://downloads.intercomcdn.com/i/o/h2osv3w3/1743851245/c389a16cb2535a1078a463ce7016/Screenshot+2025-09-25+at+12_05_22.png)
