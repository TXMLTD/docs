title: "TextMine — 16 October 2025 release"
description: "Adds folders in Vault for organising documents, with filtering, sorting, a Folders tab, moving documents between folders, and carry-through to Legislate."
---

The 16 October 2025 release introduces folder organisation in Vault, enabling documents to be grouped, filtered, sorted, and moved between folders — with folder structure carried through to Legislate.

## Folders in Vault

When importing documents into Vault, you can now organise them in folders. Select an existing folder or create a new one.

![Folder selection during document import](https://downloads.intercomcdn.com/i/o/h2osv3w3/1781103132/0e3403c84b87b3251123dc7a802e/Screenshot+2025-10-16+at+10_55_40.png)

After the documents have been imported into a folder, you can add the folder column to the view and filter the Vault by that folder.

![Adding the folder column to the Vault view](https://downloads.intercomcdn.com/i/o/h2osv3w3/1781108441/ccd693ee99f481fec0c253f4e335/Screenshot+2025-10-16+at+10_57_53.png)

![Filtering the Vault by folder](https://downloads.intercomcdn.com/i/o/h2osv3w3/1781108204/65810c0498331eb7f052c4602871/Screenshot+2025-10-16+at+10_58_15.png)

You can also move documents between folders by selecting them and clicking the **Move to folder** icon. You will then be prompted to confirm which folder you would like to send the documents to.

![Moving documents to a different folder](https://downloads.intercomcdn.com/i/o/h2osv3w3/1781110067/ddea7c7df14fdf1876cd001d2791/Screenshot+2025-10-16+at+10_56_56.png)

You can view all the folders in the Vault under the **Folders** tab.

![Folders tab showing all folders in the Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1781114196/f23a256486c40afb985a506998ae/Screenshot+2025-10-16+at+11_00_56.png)

Click on a folder to see all the documents inside it. You can also edit the folder name if required.

![Folder contents view with rename option](https://downloads.intercomcdn.com/i/o/h2osv3w3/1781116255/0a6b224790cb6892dff09831d4e3/Screenshot+2025-10-16+at+11_01_10.png)

When you submit documents to Legislate, the folder structure will be carried through.
