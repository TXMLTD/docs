---
title: "TextMine — 17 July 2025 release"
description: "Adds per-Vault document type selection, a TextMine library with type promotion controls, and agent support for pulling 10-K and 20-F filings from a CSV."
---

The 17 July 2025 release gives you finer control over document type classification per Vault and extends agents to pull SEC filings for companies listed in a CSV.

## Document type improvements

It is now possible to choose which document types — including types created by TextMine — should be included in document classification.

![Document types tab showing active types for a Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625332572/242d931bd2d073cb9ef0c8fd4f6d/Screenshot+2025-07-17+at+13_39_03.png)

Your active document types are shown in the **Document Types** tab. You can enable or disable them for a specific Vault at any time.

Click on a document type to view its associated default tags.

![Default tags for a document type](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625335809/9422998185eac9dfcd5631b39a4c/Screenshot+2025-07-17+at+13_46_15.png)

Default tags can be enabled or disabled as required.

Document types from the TextMine library can be promoted to your own document types. Select the types you wish to promote and click **Promote**.

![TextMine document type library with Promote option](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625340597/e8ab3c915fc9bc0b84fcf87cc943/Screenshot+2025-07-17+at+13_39_44.png)

Once a document type is associated with a Vault, it will be used as part of the classification process. As a result, you no longer need to make special configurations in the Vault settings.

![Document type associated with a Vault in classification settings](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625344215/25c989970a0ea7cd7e3ad451dda4/Screenshot+2025-07-17+at+13_50_28.png)

## Agent improvements

Agents can now pull 10-Ks and 20-Fs for companies listed in a CSV, XLS, or XLSX file.

![Agent configuration with CSV company list upload](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625351758/31daa638ae74b8bd56aea8d2f577/Screenshot+2025-07-17+at+13_52_23.png)

Download the example CSV and complete it with company names.

![Example CSV with company names](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625352680/9a2162142e8be2cb61d9623623b8/Screenshot+2025-07-17+at+13_52_52.png)

Review the suggestions and save the agent.

![Reviewing company suggestions before saving](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625353497/16f6cfa6b216b86e99fb63e96937/Screenshot+2025-07-17+at+13_52_30.png)

This will save the companies and submit the latest 10-Ks and 20-Fs for those companies to the corresponding Vault.

![Agent submitting 10-Ks and 20-Fs to Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1625354076/4f32cde8d6767dcc7b92eea68cf8/Screenshot+2025-07-17+at+13_54_12.png)
