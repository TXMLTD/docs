The November 1, 2024 release introduces custom flows that let users bypass the default TextMine document processing pipeline in Vault.

## Custom flows available

TextMine now allows users to skip the TextMine default document processing pipeline in Vault. When uploading a new document to Vault, simply untick the **"Keep default TextMine field"** checkbox.

In this setting, TextMine will run only the enabled Vault team tags against the document.

Once the document has been processed, the type will be listed as unspecified. Click on it to view the extracted information.

<Note>
By deselecting the TextMine fields option, the model will not currently classify the document or attempt to identify signatures.
</Note>
