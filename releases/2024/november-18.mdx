title: "TextMine — November 18, 2024 release"
description: "Vault now supports document pagination with page-level navigation, the ability to jump to a field's source page in the preview, and marked-up PDF exports."
---

The November 18, 2024 release introduces document pagination and marked-up PDF exports in Vault.

TextMine has released a number of new features to improve the user experience:

- Pagination of documents in Vault
- The ability to jump to a relevant page in the preview rather than the plain text
- The ability to generate a marked-up PDF containing the extracted fields

## Pagination

Users can go to a specific page or select a field to jump to the page which contains that field. Users can also switch between the preview and plain text of a specific page.

## Marked-up exports

You can now generate a marked-up copy of the preview which can be easily downloaded and shared with relevant stakeholders.

The PDF can be generated with the download button next to the plain text toggle. The fields will be listed on the corresponding page where the model found the answer.
