---

The April 10, 2024 release resolves several bugs and brings Scribe out of beta with custom template support.

## Bug fixes

TextMine has released the following bug fixes:

- Bugs related to the search bar
- Bugs related to the file storage integrations

## Scribe — general availability

TextMine has released Scribe from beta to general availability. The main aspect of this release is that Scribe now supports custom templates and [fields](https://help.textmine.com/en/articles/9220737-template-fields).
