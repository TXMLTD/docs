The 28 January 2026 release introduces a new Reports feature that lets you generate PDF or Word documents using data extracted from Vault.

## Reports

You can now create PDF or Word reports using data extracted from Vault.

Select the documents which should be included in the report directly in Vault and click **Generate Report**.

![Selecting documents and clicking Generate Report in Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998441380/cb79ff0be4edf72f863402e9f7f5/Screenshot+2026-01-28+at+09_07_49.png)

You will then be able to create your report request. You can either use an existing template or create a new request from scratch.

![Report request creation — choose a template or start from scratch](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998351957/303d60a1d88befb6de471b9ce888/Screenshot+2026-01-28+at+08_41_31.png)

![Report request editor](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998353249/e1ffa473b9fdd87866019f806f60/Screenshot+2026-01-28+at+08_43_17.png)

You can also save your report request as a template for future reports. Click **Generate Report** and the report will start processing.

You can access reports from the reports icon in Vault.

![Reports icon in Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998409859/d3c770175276442b969e1952b961/Screenshot+2026-01-28+at+08_59_35.png)

![Reports list view](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998410209/28dd6deb01dbff10a0df56fac1e7/Screenshot+2026-01-28+at+08_59_37.png)

Click on a report to see the results in more detail.

![Detailed report view](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998411340/b292de6fe180bc3ea7814901b1fe/Screenshot+2026-01-28+at+08_48_11.png)

Findings can be copied or you can download versions of the report in Word or PDF.

![Download report as Word or PDF](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998403301/f70f246ac03ff9ac5ce6c073235e/Screenshot+2026-01-28+at+08_50_20.png)

![Word report output example](https://downloads.intercomcdn.com/i/o/h2osv3w3/1998402465/db22aa7cafd17e87a3c19073cf86/Screenshot+2026-01-28+at+08_51_39.png)
