title: "TextMine — 12 March 2026 release"
description: "Introduces tabular views for Vault data with configurable columns, and Open Corporates API integration for accessing company data directly in the platform."
---

The 12 March 2026 release adds tabular views for exploring Vault data and integrates Open Corporates to bring company data directly into TextMine.

## Tabular views

Data in Vault can now be viewed from the Vault team in tabular format. To activate this, click on the table icon.

![Tabular view of Vault data](https://downloads.intercomcdn.com/i/o/h2osv3w3/2156002865/917c5271858766fe948a2d877173/Screenshot+2026-03-12+at+10_07_14.png)

You can choose which columns to include and adjust additional settings for each column.

![Column selection panel](https://downloads.intercomcdn.com/i/o/h2osv3w3/2156006829/984b2f762830d57636d1c3ede270/Screenshot+2026-03-12+at+10_07_38.png)

![Column settings options](https://downloads.intercomcdn.com/i/o/h2osv3w3/2156006390/16eac7288c2288b20b197a544a56/Screenshot+2026-03-12+at+10_08_47.png)

## Open Corporates

You can now access open corporate data in the platform with your Open Corporates API key.

![Open Corporates API key configuration](https://downloads.intercomcdn.com/i/o/h2osv3w3/2156009258/79e6faaaa298fdd6d441029ebbf4/Screenshot+2026-03-12+at+10_09_54.png)
