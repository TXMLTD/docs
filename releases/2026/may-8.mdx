The 8 May 2026 release introduces Ask Vault, a major new capability for querying and acting on your Vault data through a chat interface.

## Ask Vault

Ask Vault is a new feature which allows you to interrogate, transform and analyse the data extracted by Vault. You can invoke it by pressing `Q` on your keyboard or by clicking the **Ask Vault** button.

You can then ask questions about the documents, reports and agents in the vault.

![Ask Vault chat interface](https://downloads.intercomcdn.com/i/o/h2osv3w3/2364919706/0777adc610c23aa7ac95340b4ca4/Screenshot+2026-05-08+at+10_09_57.png)

Within the chat you can:

- Generate tables which can then be exported to CSV/XLSX
- Generate exportable plots from those tables
- Query failed documents
- Assign/unassign documents
- Reprocess or change a document's type
- Run reports or agents

![Ask Vault actions panel](https://downloads.intercomcdn.com/i/o/h2osv3w3/2364938906/8104bd4233174c992cbf378fb4af/Screenshot%2B2026-05-08%2Bat%2B10_09_46.png)

Each answer is presented with source documents in the Vault to facilitate cross-referencing. Steps requiring an action (for example, assignment) will ask you to confirm before performing the action.

![Ask Vault answer with source documents](https://downloads.intercomcdn.com/i/o/h2osv3w3/2364954584/4a60072110b167a9944f6aa02236/Screenshot+2026-05-08+at+10_22_34.png)

![Ask Vault confirmation prompt](https://downloads.intercomcdn.com/i/o/h2osv3w3/2364954936/9b24c136c03f32107b0b6919bc89/Screenshot+2026-05-08+at+10_22_49.png)

You can pop out the chat into a new window for better visibility.

![Ask Vault pop-out window](https://downloads.intercomcdn.com/i/o/h2osv3w3/2364921993/31f8e48584383ab2e832c0fc838c/Screenshot+2026-05-08+at+10_08_30.png)

<Note>
  Ask Vault chats are unique to each user and are scoped to the vault they are in. This first version does not have chat history — chats are lost when you refresh the page or switch screen or team.
</Note>
