---
title: "TextMine — 19 January 2026 release"
description: "Adds image document types, manual and automatic table extraction, chained tags in document types, pop-out tag cards, zoom controls, and editable agent actions."
---

The 19 January 2026 release is one of the largest updates to date, introducing image document support, table extraction, chained tags in document types, pop-out tag cards, zoom controls, expanded analytics, and editable agent actions.

## Image types

You can now create document types for images — Vault now supports image uploads (JPEG, PNG). To do this, create a document type and select the image file type. You can associate the image type to the relevant Vaults.

![Creating an image document type](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962413509/d3673ee52082b37399639cc87e06/Screenshot+2026-01-19+at+12_43_17.png)

You can then create default tags for the image.

![Default tags for an image document type](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962413040/f4501c6b8de83ade75a931d0daa1/Screenshot+2026-01-19+at+12_45_46.png)

Once the image tags are created, you can upload the image to the corresponding Vault.

![Uploading an image to Vault](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962413798/f708692b60ac8a159f2ed02aa2ee/Screenshot+2026-01-19+at+12_47_03.png)

You can then review the image and associated tags.

![Reviewing the image and its tags in the editor](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962412692/2c2ecb397098cc73d50a740c176a/Screenshot+2026-01-19+at+12_48_03.png)

## Table extraction

TextMine now offers table extraction from documents. You can either extract tables via a manual selection or enable it automatically for the Vault.

### Manual selection

Click on **Extract table from image** and use the cursor to select the table in the preview.

![Selecting a table region in the document preview](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962389471/cb81f78af4503b77f7429e9a1eca/Screenshot+2026-01-19+at+12_41_41.png)

![Table selection confirmed](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962389808/69d862b05664284474ef265194f5/Screenshot+2026-01-19+at+12_41_32.png)

Once the extraction is complete, you will see the table in the **Table** tab. You can edit the values or download the extracted table to CSV.

![Extracted table in the Table tab](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962387148/085cfa0e3da294d6878bdf3e353c/Screenshot+2026-01-19+at+12_32_33.png)

To cross-reference the results more easily, click the pop-up arrow above CSV to open the table in a new window.

![Table popped out in a new window](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962385866/9069e15fc2f9690be3e9e35c8acc/Screenshot+2026-01-19+at+12_33_02.png)

To enable automated table extraction for the Vault, go to the **Vault settings** and enable it under **Default settings**. This will apply to all documents uploaded to the Vault.

![Enabling automated table extraction in Vault settings](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962392320/1eb5a7419d407d2774c4ed5109f4/Screenshot+2026-01-19+at+12_42_28.png)

## Pop-out tags

![Pop-out tags button in the editor](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962435835/c92c1238b0c62ab69249e2ad34da/Screenshot+2026-01-19+at+12_55_53.png)

The same functionality as in the editor is mirrored in the pop-out.

![Tags in pop-out window](https://downloads.intercomcdn.com/i/o/h2osv3w3/1962435170/9ce8cf240c52d489f860ccf22a07/Screenshot+2026-01-19+at+12_56_16.png)
